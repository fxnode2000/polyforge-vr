Hand-tracked polygonal modeller · Meta Quest 3

Gestures
👌 Pinch right hand → select / drag
✊ Fist right hand → cycle mode (Object › Vertex › Face)
✊ Fist left hand → spawn new cube
👌👌 Both pinch + pull apart → extrude face

A debug HUD floats above your left wrist showing
live pinch distance and detected gestures
