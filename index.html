POLYFORGE VR

Hand-tracked polygonal modeller · Meta Quest

WebXR not available — open in Quest Browser
Gestures:
Pinch (index+thumb) · hover over geometry to highlight · pinch to select
Two-hand pinch + spread apart → extrude face
Pinch + hold on vertex/face → drag to move
Fist with right hand → cycle edit mode
Fist with left hand → add new cube
