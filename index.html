POLYFORGE VR

Hand-tracked polygonal modeller · Meta Quest 3

WebXR not supported — open in Quest Browser
Palette floats above left wrist · tap with either hand

Object mode: either hand pinch → select & drag · both pinch → grab (move+rotate+scale)
Vertex mode: either hand pinch near vertex → select & drag
Face mode SELECT: either hand pinch near face → select · both pinch → grab
Face mode EXTRUDE: both pinch selected faces + pull → extrude

Either fist → add cube · both fists → delete selected
