POLYFORGE VR

Hand-tracked polygonal modeller · Meta Quest 3

WebXR not supported — open in Quest Browser
Mode buttons float above your left hand — tap with right index finger to switch

Right hand pinch gestures:
Pinch near vertex/face → select
Pinch + hold + move → drag
Both hands pinch + pull apart → extrude face
Right fist → add new cube
Left fist → delete selected
